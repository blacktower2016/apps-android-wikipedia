Домашнее задание №20. Именование элементов

1. Задание выполняем в homeworks/homework20
2. Воспроизвести финальный код с урока для организации хранения имён.
3. Создать класс с шагами NamedSteps для шагов и реализовать шаг для клика
и для проверки видимости, который принимает только элемент без имени,
а в текст шага автоматически проставляет имя через getName
4. Создать поле namedSteps в TestContextExt для автоматического создания класса с шагами.
5. Добавить имена для экранов, ресайклеров, методов получения блока по имени или индексу,
\в элементы экранов и блоков - которые будут затронуты в следующем тесте.
6. Написать тест, который будет переходить на Explore, на следующий экран через
подготовленный блок и там куда-нибудь перейти или что-нибудь проверить на отображение.
Цель - получить в отчёте шаги с иерархией имён.
